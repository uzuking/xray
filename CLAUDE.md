# xray プロジェクト

X線イメージング研究（NanoTerasu等）のデータ管理・解析リポジトリ。

## 絶対ルール

- **ファイル削除禁止**: ユーザーの明示的な許可なく、いかなるファイルも削除しないこと
- **Python管理**: 必ず `uv` で管理し、`uv` で実行すること（pip/conda/venv は使わない）

## データ管理

- 大容量データ（CT画像、実験写真等）は **DVC** で管理し、Dropbox にバックアップ
- DVC リモート: `~/Dropbox/dvc-storage/xray`
- Git には `.dvc` ファイルのみコミットし、実データはコミットしない

## ディレクトリ構成

```
NanoTerasu/
  260207_Komatsuna/     # 2026/2/7 BL09W 白色CT コマツナ根のCTデータ
    rec/                # 再構成スライス画像 (TIF)
  exp_photos/           # 実験時の写真
  Emailのやり取り/       # 利用照会〜実験調整のメール記録
```

## NanoTerasu 実験概要

- 施設: NanoTerasu（仙台）BL09W 白色CT
- 日時: 2026年2月7日
- サンプル: コマツナの根
- 空間分解能: 約5 um
- X線エネルギー: 白色X線 4-30 keV（20 keVピーク）
- 共同研究者: 大西弘志先生、吉本則之先生（理工学部）
