# NanoTerasu利用の経緯

## 概要

2026年2月7日（土）、仙台のNanoTerasuにてコマツナ根のX線CT撮影を実施。
理工学部・大西弘志先生チームの実験枠に農学部・松嶋が参加する形で実現した。

## 施設・ビームライン情報

| 項目 | 内容 |
|---|---|
| 施設 | NanoTerasu（仙台） |
| ビームライン | BL09W 白色CT |
| 測定日 | 2026年2月7日（土）10:00-18:00 |
| サンプル許容サイズ | 直径 20 mm 以内 |
| 測定視野 | 20 mm x 4 mm |
| 空間分解能 | 約 5 um |
| X線エネルギー | 白色X線 4-30 keV（20 keVにピーク） |
| 測定時間 | 15分 / 1測定 |
| 費用 | ビームライン使用料は全学負担、旅費は各研究室負担 |

## 時系列

### 2026年1月9日 — 照会開始
山谷（研究・地域連携課）が農学部の松嶋・折笠に参加を照会。
理工学部・大西チームの実験で計測時間に余裕があるため、サンプルがあれば一緒に測定できるという話。

### 2026年1月13日 — 折笠先生辞退
折笠先生が日程・サンプル準備の都合で辞退。次回参加を希望。

### 2026年1月16日 — 松嶋検討中
松嶋が電話で「迷っている」と伝達。別件でサンプルを育てており、
ついでにCT用サンプルを準備できるか検討中。

### 2026年1月26日 — 松嶋一旦辞退→吉本先生の呼びかけ
松嶋が「良いアイディアが浮かばない」として一旦辞退。
しかし吉本先生が「研究室に転がっているもので気軽に撮ってみては」と再考を促した。
吉本先生自身は装置性能評価のためにコーヒー豆を撮影する予定とのこと。

### 2026年2月3日 — コマツナ根の観察を希望
松嶋が「コマツナの根を観察したい」と具体的な希望を表明。
サンプル固定法の懸念を相談。

### 2026年2月4日 — 技術的打ち合わせ
吉本先生が「直径20mm x 高さ4mmに収まればOK、15分間固定すれば良い」と助言。
同日16:30にZoomミーティングを実施し、具体的な段取りを決定。
藤崎さんもZoom参加。

### 2026年2月5日 — サンプル準備（推定）
exp_photos内の写真の日付から、この日にサンプル準備・撮影を行った可能性あり。

### 2026年2月7日 — NanoTerasuで測定実施
260207_Komatsuna ディレクトリのデータから、予定通り測定が行われた。
